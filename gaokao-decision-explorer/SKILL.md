# 🎯 高考志愿决策探索器 — 多维度问询

> 基于 Prism Discover 框架改造，将高考志愿决策拆解为19个分析维度。
> 每个维度：提出关键问题 → 揭示隐蔽影响 → 给出方向提示

## 使用方式

加载本 skill 后，**先收集考生个人情况** → 再逐维度问询 → 最终输出决策画像+矛盾点+推荐方案。

### 数据引用规则
- **知识库优先**：每个维度的数据依据优先从 `references/数据源索引.md` 映射的知识库文件读取
- **知识库不够 → web_search**：如果知识库文件不包含所需信息，立即调用 web_search 补充
- **过期检测**：知识库数据超过3个月的，用 web_search 交叉验证后使用

### 交互规则（必遵守）
- **一次只问一个问题**，用户回答后再问下一个。严禁一次抛出多个问题。
- 步骤0的个人情况也要逐项提问，不要一次列出所有字段。
- 每个维度内部的问题也逐个问，不要一次列3个问题。
- 已回答的信息在后续维度中不再重复提问。
- 回答简洁，陷阱提醒和方向提示在用户给出答案后再展示。

## 📋 步骤0：考生个人情况

在执行19个维度问询之前，必须先收集以下基本信息。根据用户的回答情况灵活补充。

### 基本信息
- **省份**：（必填）高考所在省份
- **选科**：（必填）物理类 / 历史类 / 3+3具体科目
- **分数 / 位次**：（选填）出分前填"估分约XX"，出分后必填
- **性别**：男 / 女
- **应届/复读**：应届 / 复读

### 家庭背景
- **家庭经济情况**：急需工作补贴家用 / 能支持读完大学 / 能支持读研 / 资源雄厚
- **家庭行业人脉**：有（具体行业） / 没有
- **父母期望**：稳定工作 / 高收入 / 孩子自己决定 / 其他

### 方向倾向（初步）
- **有没有特别想去的城市？** 武汉 / 北上广深 / 其他 / 没偏好
- **有没有特别感兴趣的专业方向？** 工科 / 医学 / 法学 / 师范 / 经管 / 没想法
- **对体制内工作的态度？** 想进 / 不想进 / 看情况
- **计划读研吗？** 必须读 / 看情况 / 不想读

### 个人条件
- **身体条件**：色盲色弱 / 近视度数 / 其他体检限制
- **最擅长的科目**：帮助判断能力倾向

> **注意**：这些信息将在19个维度中反复引用。对于已回答的信息，后续维度中不再重复提问。

## 最终输出

所有维度问答完成后，输出：
1. **决策画像** — 基于各维度回答的用户偏好画像
2. **矛盾点标记** — 维度间回答冲突的警示（如：想留武汉但选了只有一线有好工作的专业）
3. **推荐方案** — 基于画像的3个候选方案（冲/稳/保）

## 维度1：⚡ 就业薪资与岗位供需

### 关键问题
- 你希望毕业第一年年薪达到多少？6-8万 / 10-15万 / 20万+ / 无所谓
- 你能接受毕业后花几年时间在低收入阶段积累经验吗？3年 / 5年 / 不能接受低收入
- 你更看重起薪还是长期薪资增长潜力？短期快钱 / 长期曲线 / 两者都要

### 隐蔽的陷阱
> 📌 起薪高的专业（计算机/AI）35岁后可能面临红线；起薪低的专业（医生/律师）越老越值钱
> 📌 **薪资不等于可支配收入**——一线城市月薪2万 vs 三线月薪8千，实际生活质量可能后者更高
> 📌 行业均值掩盖了企业梯队差异——同一专业顶尖大厂 vs 外包薪资差距可达3-7倍

### 方向提示
- 🟢 急需赚钱养家 → 计算机、电气工程、口腔医学
- 🟡 能接受先苦后甜 → 临床医学、法学（过法考进体制）、会计（考CPA）
- 🔵 追求稳定增长 → 电气工程（进国网）、师范（考编）、公务员（法学）

## 维度2：🏛️ 体制内通道（考公/考编/选调生）

### 关键问题
- 你计划考公务员或进事业单位吗？是 / 否 / 不确定
- 你对选调生有了解吗？清楚 / 听说过 / 第一次听说
- 你能接受去基层工作2-3年作为跳板吗？可以 / 不太想 / 完全不能
- 你更倾向体制内的稳定性还是企业的高收入？体制内 / 企业 / 不知道

### 隐蔽的陷阱
> 📌 选专业时觉得"考公不限专业"——实际上2025国考仅22个岗位不限专业（0.11%），95%岗位有专业限制
> 📌 **法学考公最优**不是因为它名额最多，而是因为专业壁垒高（需过法考），竞争者相对少，进面分反而最低
> 📌 选调生是文科生进体制的VIP通道——竞争比20:1 vs 省考98:1

### 方向提示
- ✅ **考公优先** → 法学（40%+岗位）> 会计/审计 > 汉语言 > 公共管理
- ✅ **追求选调生** → 优先选能上武大华科的分数（中央选调仅30所高校有资格）
- ✅ **求稳不求高薪** → 师范（考编）> 公安类 > 税务相关专业

## 维度3：🔮 4年后行业景气预测（2030年毕业）

### 关键问题
- 你更相信传统稳定行业还是新兴赛道？传统 / 新兴 / 结合
- 你是否关注国家政策方向？关注 / 不关注 / 略知一二
- 你能接受所选行业4年后走下坡路的风险吗？能接受 / 不能接受 / 会提前规划

### 隐蔽的陷阱
> 📌 现在的热门≠4年后还热。2021年土木工程还是高分专业，2025年237家设计院关门
> 📌 **最确定的不是某个行业，而是政策驱动的结构性趋势**：AI国产替代、双碳新能源、人口老龄化
> 📌 有些专业需求是刚性的（电气/医学/法学），不受经济周期影响太多

### 方向提示
- 🔥 **确定性最高的增长赛道** → AI/芯片（国产替代10年主线）、新能源/储能（双碳2030）、军工（每年稳定增长）
- 📉 **风险最高的下行行业** → 土木/建筑（需等1-3年复苏）、传统媒体（AI替代）、基础编程（AI替代85%）
- ✅ **攻守兼备** → 电气工程（进可新能源汽车30W，退可国网15W）

## 维度4：🧬 转行弹性（退出壁垒）

### 关键问题
- 你对所选专业的忠诚度有多高？认定不回头 / 可能会转行 / 不确定
- 如果毕业后发现不喜欢这行，你有多大能力转行？很有把握 / 有点担心 / 完全没底
- 你能接受本科4年的学习投入后转行到不相关领域吗？能 / 有点浪费 / 不能接受

### 隐蔽的陷阱
> 📌 生物工程转行率83%、环境工程79%、化学工程77%——这些专业本科毕业几乎没对口工作
> 📌 **转行难度和专业壁垒成正比**——临床医学壁垒最高（几乎无法转行），但薪资也最稳
> 📌 看似"好就业"不一定"好转行"——计算机薪资高但非科班极难转入

### 方向提示
- 🔓 **易转行** → 工商管理、市场营销、汉语言（什么都能干，但啥都不精）
- 🔒 **难转行** → 临床医学（8年培养=沉没成本）、电气工程（技术壁垒高）
- ⚠️ **被迫转行率最高** → 生物工程83%、环境工程79%、化学77%（×避免）

## 维度5：🧭 城市与地域绑定度

### 关键问题
- 你未来想留在哪个区域？湖北 / 北上广深 / 长三角 / 珠三角 / 成都重庆 / 无所谓
- 你能接受去一线城市打拼吗？想挑战 / 不太想 / 完全不想
- 你对房价和城市生活成本有概念吗？清楚 / 大概知道 / 没概念
- 你更看重城市发展机会还是离家近？看机会 / 离家近 / 两者平衡

### 隐蔽的陷阱
> 📌 有些专业**必须去一线才有机遇**：AI算法55%岗位在京沪深，金融核心岗在沪京深
> 📌 有些专业**在哪都能活**：电气工程（各地电网+设备厂）、会计、教师、护理
> 📌 **武汉的性价比**：应届生月薪7800 vs 深圳9500，但房价1.5万/㎡ vs 5.6万/㎡——实际生活质量武汉碾压

### 方向提示
- 🏙️ **想去一线打拼** → 计算机/AI、金融、芯片设计、新媒体
- 🏡 **想留湖北/二三线** → 电气工程（电网）、医学（三甲省市各层级）、师范（考编）、会计
- 🔄 **灵活切换** → 计算机（一线赚够钱可远程或回二线）、医学（一线学成回省市是常见路径）

## 维度6：🧩 学历通胀与深造必要性

### 关键问题
- 你计划读研吗？必须读 / 看情况 / 不想读 / 不确定
- 你家庭的资金能支持你读研/读博吗？能支持 / 有点紧张 / 需要尽快工作
- 你能接受5+3+3（临床11年）或2-4年（硕博）的长期投入吗？能 / 不能 / 看专业值不值

### 隐蔽的陷阱
> 📌 有些专业**本科等于没读**：生物工程83%转行、材料化学不读到博没出路
> 📌 有些专业**本科完全够用**：电气工程进国网本科即可，计费机本科可拿40W，护理本科进三甲
> 📌 学历通胀在加速：10年前本科能进的大厂，现在要硕士+985起步

### 方向提示
- 🎓 **本科就业即可** → 电气工程、计算机（本科也能拿高薪）、护理学、会计（+CPA）
- 📚 **必须读到硕博** → 临床医学（博士进三甲）、生物/材料/化学（不读博没出路）、法学（五院四系硕进红圈所）
- ⚠️ **考研内卷最严重** → 计算机（考录比20:1+）、法学（考研率43%）、临床医学（专硕竞争激烈）

- 你的预估分数大概能上什么层次的学校？武大华科线 / 武理中南财线 / 一本线 / 二本线
- 你对大厂校招有了解吗？清楚 / 听说过 / 完全不了解

### 隐蔽的陷阱

### 方向提示
- 🏆 **武大华科线** → 所有企业target全覆盖，选什么专业都有好出路
- 🥈 **武理/中南财/华师** → 部分企业target，选对专业（计算机/电气/法学）可以突破
- 🥉 **一本线** → 靠专业突围（电气工程进国网、护理进三甲、会计考CPA）
- 🟢 **双非** → 选有硬门槛的专业（电气/护理/会计），证书+实习=逆袭路径

## 维度8：🧮 证书杠杆效应

### 关键问题
- 你愿意在大学期间花额外时间考职业资格证书吗？非常愿意 / 看情况 / 不想考 / 没概念
- 你能接受考证可能需要2-4年持续投入吗？能 / 有点犹豫 / 不想
- 你了解各专业的核心证书吗？了解 / 听说过一些 / 完全不了解

### 隐蔽的陷阱
> 📌 法学本科毕业：不过法考=月薪5000，过法考=进体制月薪1万+，考公优势极大
> 📌 CPA持证人：月薪跃升50-175%，四大合伙人必备
> 📌 **有些专业无证可考**：工商管理、市场营销、公共管理等没有核心职业证书，0杠杆

### 方向提示
- 🏅 **证书是命根子** → 法学（法考）、会计（CPA）、电气（注电）、医学（执医）
- 📜 **证书有加分但不是必须** → 计算机（软考高级）、金融（CFA）、英语（CATTI/雅思）
- ❌ **无证可考的专业** → 工商管理、市场营销、公共管理——这类专业就业只能拼实习和学校品牌

- 💪 **急需挣钱** → 计算机（3年年薪可达40W+）、电气工程（毕业即15W+）、护理（3年可达10-15W）

## 维度10：🧠 AI替代脆弱性

### 关键问题
- 你用AI工具做过什么事？经常用 / 试过 / 没用过
- 你选择专业时会考虑AI会不会替代吗？考虑了 / 没考虑 / AI离这行很远
- 你更倾向于做"与AI协作"还是"逃避AI"的工作？与AI协作 / 做AI替代不了的 / 没想过

### 隐蔽的陷阱
> 📌 AI不是消灭所有岗位，而是**消灭中间层**——初级程序员替代率85%，高级架构师反而更值钱
> 📌 纯机械性脑力工作最危险：基础翻译（替代90%+）、基础会计（替代80%）、初级设计（替代70%）
> 📌 动手+安全责任+情感陪伴=AI难以替代的铁三角：医生/护士/电工/警察/心理师

### 方向提示
- 🛡️ **AI最难替代** → 临床/口腔（动手+信任）、电气工程（带电作业=安全责任）、护理（情感陪伴）
- 🤖 **需要拥抱AI** → 计算机+AI（不是学编程，是学"用AI编程+设计AI系统"）、AI+行业应用
- ⚠️ **被AI替代风险高** → 基础翻译、基础会计、基础设计、基础编程、客服

## 维度12：🎭 个人兴趣与能力匹配

### 关键问题
- 你高中最喜欢/最擅长的科目是？（提示：这个信息能帮你判断适合什么方向）
- 你更偏好什么类型的工作？和机器打交道（技术型）/ 和数字打交道（分析型）/ 和人打交道（社交型）/ 和文字打交道（创意型）
- 你能接受加班/出差/倒班/下工地/值夜班等工作方式吗？能接受某几种 / 完全不能 / 看薪资

### 隐蔽的陷阱
> 📌 兴趣≠能力：喜欢打游戏≠适合学计算机，喜欢动物≠适合学兽医
> 📌 **大学课程内容和高中完全不同**——很多学生因为大学的专业课程和自己想的不一样而退学/转专业

### 方向提示
- 🔬 **喜欢动手/实验** → 临床医学、口腔医学、电气工程（实训多）
- 💻 **喜欢逻辑/数学** → 计算机、电子信息、数学/统计（数据分析方向）
- 📝 **喜欢文字/表达** → 法学、汉语言、新闻传播（需叠加技能）

## 维度16：⏳ 时间复利曲线

### 关键问题
- 你愿意为职业发展投入多长的积累期？3年 / 5年 / 10年 / 越长越好
- 你更偏好"快回报"还是"高顶峰"？快回报（25岁月入1万+，后面缓慢增长）/ 高顶峰（前5年低薪，45岁达到职业巅峰）
- 你对35岁之后的发展有规划吗？有想法 / 没想那么远 / 认为专业决定后半生

### 隐蔽的陷阱
> 📌 医生的薪资曲线是标准的"J型"：前8年低收入（规培月薪3000），45岁后年薪50万+，70岁还能返聘
> 📌 程序员的薪资曲线是"倒U型"：25岁年薪40万，35岁面临红线，之后急剧下降
> 📌 电气工程师是"平稳上升型"：25岁15万→40岁30万→55岁还稳定——虽然顶峰不高，但稳

### 方向提示
- 📈 **高顶峰慢回报** → 临床医学（50岁达到巅峰）、法学（资深律师年薪百万）、会计（CPA持有者越老越值钱）
- 🚀 **快回报低后劲** → 计算机（35岁红线）、互联网运营（吃青春饭）、设计（容易被年轻替代）
- 🛤️ **稳步增长** → 电气工程（电网）、师范（编制+教龄）、公务员（职级+年限）、护理（经验积累）

## 维度19：📋 志愿填报策略操作层面

### 关键问题
- 你了解湖北的志愿填报规则吗？院校专业组是什么、45个平行志愿怎么排序、梯度怎么分配？完全明白 / 大概知道 / 完全不懂
- 你更倾向于"冲学校"还是"保专业"？冲学校（进去再转专业）/ 保专业（直接录到想要的）/ 折中
- 你打算服从专业调剂吗？服从 / 不服从 / 看情况

### 隐蔽的陷阱
> 📌 同一学校不同专业组之间分数差距可达50分——武理工最高组632分 vs 最低组583分
> 📌 冲学校不服从调剂=退档风险极大（一旦退档只能等征集志愿或下一批次）
> 📌 45个平行志愿不是45个"都想去的学校"——合理结构是冲10/稳20/保15

### 方向提示
- 🚀 **冲学校策略** → 适合对学校层次有执念、能接受冷门专业、有转专业计划的学生
- 🎯 **保专业策略** → 适合有明确职业方向、不能接受冷门专业、不想冒险的学生
- 📐 **推荐比例** → 冲10-15个 / 稳15-20个 / 保10-15个（湖北45个志愿）
